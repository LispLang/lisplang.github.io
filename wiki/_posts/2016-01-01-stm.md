title: Software Transactional Memory

**Software Transactional Memory** is a mechanism for managing memory in a
concurrent environment.

# STMX

[STMX] is a high-performance software transactional memory library for Common
Lisp. It was presented at the European Lisp Symposium 2014 ([Slides], [video]).

[STMX]: https://github.com/cosmos72/stmx
[Slides]: http://www.european-lisp-symposium.org/ghilardi.pdf
[video]: http://medias.ircam.fr/xcc8494
